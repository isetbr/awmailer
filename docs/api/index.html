M4A1 Mailer API

M4A1 is a API service to send massive mail campaigns from iSET services.

To use this API you must perform HTTP request on following methods described below, all HTTP call's must be specified a Content-Type header with value application/json, not specifying this method returns a Bad Request response and nothing is processed.

All values such as service keys, tokens, campaign keys, ip addresses used in each requests, responses and headers in following methods are examples, when you use this API new values will be generated and returned in calls, not forget to replace this values on next requests.

For all API calls, your ip address must be previous allowed in API, for the campaign methods, you will need an service key and token.
